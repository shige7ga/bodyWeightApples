# リンゴで体重管理

### リンゴの個数で、日々の体重変化を記録・管理できるアプリ

### 公開URL：https://applebodyweight.onrender.com/

### GitHub：https://github.com/shige7ga/bodyWeightApples

## ミニアプリを作成した理由
- 「自分の体重はリンゴだと何個分だろう？」という興味を引くフックを通じて、体重管理を気軽に始められるようにしたかった
- kgという単位は直接的でプレッシャーを感じやすいため、リンゴ何個分という柔らかい表現にすることで、心理的ハードルを下げたかった
- 目標体重までを「あとリンゴ何個分」と可視化することで、達成までの距離を直感的に理解できるようにしたかった
- リンゴ何個分というユニークな表現により、体重管理の記録をSNSなどで共有しやすくし、楽しみながら継続できる仕組みにしたかった

## できること（機能一覧）
- ユーザー機能
  - ユーザー登録
  - ユーザー情報の編集
  - ユーザーの削除
  - ユーザー一覧・詳細表示
  - ログイン・ログアウト機能
  - 登録された身長から理想体重を算出して表示

- 体重記録機能
  - 体重記録の作成・編集・削除
  - 記録の一覧表示・詳細表示
  - 体重を「リンゴ何個分」に変換して表示
  - 理想体重まで「あとリンゴ何個分」と表示

- コミュニケーション機能
  - 体重記録へのコメント投稿・編集・削除

## 今後の改善ポイント
- UI/UX改善
  - 「リンゴ何個分」の表示を強調し、アプリのコンセプトが一目で伝わるUIに改善する
  - リンゴのイラストを用いて、視覚的に「何個分か」を表現する
  - 直感的に操作できるよう、UI/UX全体を改善する（導線・配置・余白など）
  - ページネーションを導入し、一覧画面の可読性と操作性を向上させる

- 機能拡張
  - ユーザーのフォロー機能を追加し、他ユーザーとのつながりを作る
  - リンゴ以外の単位（例：バナナ、ペットボトルなど）でも換算できる機能を追加する
  - ログイン前でもリンゴ換算を体験できるようにし、ユーザー獲得につなげる
  - 体重の目標値を設定し、「目標まであとリンゴ何個分」を表示できるようにする

- 継続・可視化機能
  - 記録日数のカウント機能を追加し、継続状況を可視化する
  - 体重の推移をグラフで表示し、日々の変化を視覚的に把握できるようにする
  - 記録忘れ防止のリマインド通知機能を追加する

- デバッグ・テスト
  - 開発効率向上のため、pry-byebug を用いたデバッグ手法を活用する
  - RSpecを用いたテストコードを導入する（モデル・コントローラの単体テストを想定）
  - 正常系・異常系を考慮したテスト設計をする
